#Pitstop

## A Jewerly Repair Intake App


### How to run:

```
1. Clone Repo
2. cd lib/
3. npm install,
4. bower install
5. From Lib run 'Gulp',
6. cd back to root

`http-server -p 9999`

```

This will show at:
`http://localhost:9999`
